# PHP DevOps Project

This project demonstrates a complete DevOps workflow for deploying a PHP application using modern DevOps tools.

## Technologies Used

- AWS EC2
- Docker
- GitHub
- Terraform
- Jenkins (CI/CD)
- Kubernetes (Upcoming)

## Project Architecture

Developer → GitHub → Jenkins → Docker → Kubernetes → Application

## Application

Simple PHP application that displays:

Welcome Syed Shahabuddin  
My PHP DevOps Application is Running 🚀

## Infrastructure

Infrastructure is created using Terraform.
